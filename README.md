# Project: Blogify Application

# Description:

Developed a robust blogging platform using Node.js, Express.js, MongoDB, EJS, and Bootstrap. Implemented user authentication and authorization using JWT.

# Technologies: 
Node.js, Express.js, MongoDB, EJS, Bootstrap, JWT, Mongoose.

# Database Integration:

Utilized MongoDB for data storage, managing user accounts, blog posts, and comments.

Established a connection to the MongoDB database using Mongoose, ensuring efficient data operations.

# User Authentication:

Implemented user registration and login functionality using JWT for secure authentication.

Created middleware to check for authentication cookies, ensuring that only logged-in users can add comments to blog posts.

# Dynamic Templating:

Employed EJS as the templating engine to render dynamic content on the server side.

Set up views and partials for consistent and reusable HTML structures across different pages.

# Responsive Design:

Used Bootstrap for responsive and mobile-friendly UI design.

Ensured that the application is visually appealing and functional across various devices and screen sizes.

# Routing and Middleware:

Set up RESTful routes for user and blog-related operations, ensuring a clean and maintainable code structure.

Created custom middleware for handling authentication and authorization.

# Core Features:

Blog Creation and Management: Enabled users to create, edit, and delete blog posts.

Commenting System: Allowed authenticated users to add comments to blog posts, enhancing user interaction.

Homepage: Displayed a list of all blog posts with user information and comments.

# Cookie Handling:

Utilized cookie-parser middleware to handle cookies efficiently, managing user sessions and authentication tokens.

# Static Files Management:

Served static files such as CSS, JS, and images using Express's static middleware, ensuring proper resource management and performance.
